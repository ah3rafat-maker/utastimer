إعداد Firebase المطلوب لهذه النسخة
=================================

هذه النسخة تستخدم Cloud Firestore، وليس Realtime Database.

المجموعات/المستندات المستخدمة تلقائيًا:

settings / config
- password
- semester
- academicYear
- fileName
- updatedAt

examData / main
- rows
- rowsJson
- updatedAt
- fileName

القواعد الحالية في Firestore مناسبة للتجربة إذا كانت تسمح بالقراءة والكتابة.
بعد التأكد من عمل الموقع، يجب تشديد القواعد لاحقًا.

إذا ظهرت رسالة: "متصل بالمزامنة السحابية عبر Firestore" فهذا يعني أن الاتصال يعمل.
إذا لم تظهر، تحقق من:
1. وجود firebase-config.js.
2. وجود سكربت firebase-firestore-compat.js في index.html.
3. قواعد Firestore Rules.
4. نشر آخر نسخة على Vercel.
